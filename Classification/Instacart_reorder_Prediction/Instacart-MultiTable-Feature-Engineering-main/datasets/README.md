# Dataset Information

This project uses the Instacart Online Grocery Shopping Dataset.

Due to GitHub file size limits, the original CSV files are not included in this repository.

## How to obtain the data
Please download the dataset from the official source:
https://www.kaggle.com/c/instacart-market-basket-analysis/data

After downloading, place the following files into the `datasets/` directory:
- orders.csv
- order_products__prior.csv
- order_products__train.csv
- products.csv
- aisles.csv
- departments.csv
